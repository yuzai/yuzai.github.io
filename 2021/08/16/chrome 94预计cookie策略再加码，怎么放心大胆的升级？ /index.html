chrome 94йў„и®Ўcookieзӯ–з•ҘеҶҚеҠ з ҒпјҢжҖҺд№Ҳж”ҫеҝғеӨ§иғҶзҡ„еҚҮзә§пјҹ

chrome 94д№ӢеҗҺпјҢе°ҶзҰҒз”Ёе‘Ҫд»ӨиЎҢеҸӮж•°еҜ№SameSiteByDefaultCookiesжҸҗдҫӣпјҢеұҠж—¶пјҢжң¬ең°localhostдёӢпјҢзӣҙжҺҘи®ҝй—®жөӢиҜ•зҺҜеўғжҺҘеҸЈе°ҶжҲҗдёәй—®йўҳгҖӮжң¬ж–Үе…Ҳд»Ӣз»ҚиҝҷдёҖзӯ–з•Ҙзҡ„еҺҶеҸІеҸҠжӣҫз»Ҹзҡ„и§ЈеҶіж–№жЎҲпјҢе№¶дјҡй’ҲеҜ№жӯӨй—®йўҳпјҢжҸҗеҮәдёҖдёӘchrome 94д№ӢеҗҺзҡ„и§ЈеҶіж–№жЎҲдҪңдёәеҸӮиҖғгҖӮи®©еӨ§е®¶е°ҪйҮҸе°‘иө°ејҜи·ҜпјҢж”ҫеҝғеӨ§иғҶзҡ„еҚҮзә§chromeгҖӮ

иғҢжҷҜ

пјҲжғізӣҙжҺҘжӢҝи§ЈеҶіж–№жЎҲзҡ„зӣҙжҺҘзҝ»еҲ°и§ЈеҶіж–№жЎҲдёҖиҠӮеҚіеҸҜпјү

зӣёдҝЎеүҚдёҖж®өж—¶й—ҙпјҲ2021.5 - 7жңҲд»ҪпјҢйҖӮйҖўchrome91еҸ‘еёғпјүпјҢжңүдёҚе°‘еүҚз«ҜејҖеҸ‘еҗҢеӯҰе’ҢжҲ‘дёҖж ·гҖӮдёҖи§үиө·жқҘпјҢжҳЁеӨ©иҝҳиғҪжӯЈеёёи°ғиҜ•зҡ„йЎөйқўпјҢд»ҠеӨ©дёҖзӣҙеңЁеҸҚеӨҚжҠҘжңӘзҷ»еҪ•пјҢиҖҢиҮӘе·ұжҳЁеӨ©жҷҡдёҠдёӢзҸӯзҡ„ж—¶еҖҷиҝҳеҘҪеҘҪеҘҪзҡ„пјҢжҖҺд№ҲзқЎдёҖи§үе°ұдёҚиЎҢдәҶе‘ўпјҹ

и°ғbugзңҹзҗҶжңүдәҢпјҡ

  1. и®Ўз®—жңәжҳҜдёҚдјҡеҮәй”ҷзҡ„пјҢеҰӮжһңз»“жһңдёҚеҜ№пјҢдёҖе®ҡжҳҜд»Јз Ғй”ҷдәҶ
  2. жҺ’йҷӨдёҖеҲҮдёҚеҸҜиғҪпјҢеү©дёӢзҡ„дёҚз®ЎеӨҡд№Ҳйҡҫд»ҘзҪ®дҝЎпјҢдёҖе®ҡе°ұжҳҜзңҹзӣё

зӣёдҝЎиҒӘжҳҺзҡ„е°ҸдјҷдјҙиӮҜе®ҡеҫҲеҝ«е°ұиғҪе®ҡдҪҚеҲ°й—®йўҳпјҢз«Ӣй©¬е°ұзҘӯеҮәsafairпјҢеҸҜд»ҘжӯЈеёёдҪҝз”ЁгҖӮйӮЈд№ҲжҜ«ж— з–‘й—®пјҢchromeжӣҙж–°еёҰжқҘзҡ„(зӣёдҝЎchromeдҪңдёәеүҚз«ҜејҖеҸ‘зҡ„еҝ…еӨҮпјҢејҖеҗҜиҮӘеҠЁжӣҙж–°жҳҜеҫҲеӨҡеҗҢеӯҰзҡ„йҖүжӢ©)гҖӮиҖҢж— жі•зҷ»йҷҶпјҢж— йқһе°ұжҳҜcookieдёҚдјҡжҗәеёҰе‘—пјҢиҖҒй—®йўҳдәҶгҖӮ

йЎәеёҰдёҖжҸҗпјҡ

еңЁжҲ‘д»¬еҪ“еүҚеӣўйҳҹзӣ®еүҚзҡ„йЎ№зӣ®дёӯпјҢдёҚжҳҜжүҖжңүйЎ№зӣ®йғҪдјҡеҮәзҺ°иҝҷдёӘcookieй—®йўҳгҖӮд»”з»ҶзңӢд»Јз ҒпјҢжңүдәӣйЎ№зӣ®зҡ„жҺҘеҸЈиҜ·жұӮпјҢе§Ӣз»ҲжҳҜеҸ‘йҖҒеҲ°жң¬ең°еҹҹеҗҚзҡ„пјҢиҖҢиҜ·жұӮжөӢиҜ•зҺҜеўғпјҢжҳҜйҖҡиҝҮwebpackDevServerдёӯй…ҚзҪ®proxyе®һзҺ°жҺҘеҸЈиҪ¬еҸ‘пјҢз”ұдәҺеңЁжөҸи§ҲеҷЁзңӢжқҘе§Ӣз»ҲжҳҜеҸ‘йҖҒеҲ°localhostзҡ„пјҢжҺҘеҸЈеҹҹеҗҚжҳҜlocalhostпјҢжң¬ең°еҹҹеҗҚд№ҹжҳҜlocalhostпјҢжүҖд»ҘдёҚеӯҳеңЁcookieдёҚжҗәеёҰеҜјиҮҙж— жі•зҷ»еҪ•зҡ„й—®йўҳгҖӮ

иҖҢдёҖдәӣh5йЎ№зӣ®пјҢжҳҜзӣҙжҺҘеҸ‘йҖҒеҲ°жөӢиҜ•зҺҜеўғеҹҹеҗҚзҡ„пјҢз”ұдәҺlocalhostе’ҢжҺҘеҸЈзҺҜеўғеҹҹеҗҚдёҚеҗҢжәҗпјҢд»ҺиҖҢеҮәзҺ°дәҶcookieж— жі•жҗәеёҰпјҢд»ҺиҖҢж— жі•зҷ»еҪ•зҡ„й—®йўҳгҖӮ

еҗҢжәҗзӯ–з•Ҙзҡ„еҺҶеҸІеҸҠеҜ№еә”зҡ„и§ЈеҶіж–№жЎҲ

еҪ“йЎөйқўеҶ…еҸ‘иө·иҜ·жұӮж—¶пјҢдјҡй»ҳи®ӨжҗәеёҰиҜҘеҹҹеҗҚдёӢзҡ„cookieпјҢиҖҢcookieеҗҢжәҗзӯ–з•ҘпјҢжҳҜжҢҮпјҢйҷӨйқһеҪ“еүҚеҹҹеҗҚе’ҢиҜ·жұӮеҹҹеҗҚпјҢжҳҜеҗҢжәҗпјҢжүҚдјҡй»ҳи®ӨжҗәеёҰcookieпјҢиҝҷе°ұеҜјиҮҙпјҢlocalhostзӣҙжҺҘиҜ·жұӮжөӢиҜ•зҺҜеўғзҡ„еҖҹеҸЈпјҢжҜ”еҰӮapi.baidu.comпјҢжӯӨж—¶дёҚдјҡжҗәеёҰcookieпјҢд»ҺиҖҢеҜјиҮҙжҺҘеҸЈ301пјҢзҷ»еҪ•еӨұж•Ҳзҡ„й—®йўҳгҖӮ

иҝҷдёҖзӯ–з•Ҙд»Һchrome 80ејҖе§ӢеӯҳеңЁпјҢиҖҢchrome 91еҚҮзә§пјҢchrome 94(йў„и®Ў2021.9жңҲд»ҪеҸ‘еёғ)йў„и®ЎеҶҚеҚҮзә§гҖӮ

еңЁchrome 91д»ҘеүҚпјҢеҸҜд»ҘйҖҡиҝҮеңЁchrome жөҸи§ҲеҷЁдёӯжү“ејҖchrome://flags/пјҢи®ҫзҪ®SameSiteByDefaultCookieдёәdisableеҚіеҸҜгҖӮ

иҖҢchrome 91д№ӢеҗҺпјҢеҲ йҷӨдәҶиҜҘй…ҚзҪ®пјҢйңҖиҰҒжүӢеҠЁй…ҚзҪ®еҗҜеҠЁеҸӮж•°жқҘе…ій—ӯиҜҘйҷҗеҲ¶пјҢйҖҡиҝҮеўһеҠ еҗҜеҠЁеҸӮж•°вҖ“args вҖ“disable-features=SameSiteByDefaultCookiesгҖӮеҚіеҸҜе®һзҺ°cookieзҡ„и·Ёеҹҹдј йҖ’гҖӮиҮӘеҠЁж·»еҠ еҗҜеҠЁеҸӮж•°еҸҜд»ҘеҸӮиҖғж–Үз« гҖӮ

иҖҢchrome 94йў„и®ЎеҲ йҷӨеҗҜеҠЁеҸӮж•°зҡ„ж”ҜжҢҒпјҢйӮЈд№ҲеұҠж—¶иҜҘжҖҺд№Ҳи§ЈеҶіе‘ўпјҹ

еҮ дёӘи§ЈеҶіж–№жЎҲ

chrome 94д№ӢеҗҺпјҢеҪ“chromeзҰҒз”ЁдәҶи·Ёеҹҹcookieзҡ„е‘Ҫд»ӨиЎҢж“ҚдҪңд№ӢеҗҺпјҢз¬”иҖ…е®һи·өиҝҮзҡ„жңүд»ҘдёӢеҮ дёӘи§ЈеҶіж–№жЎҲпјҡ

еҖҹеҠ©webpack-dev-serverе®һзҺ°жҺҘеҸЈиҪ¬еҸ‘

еңЁеҸ‘иө·жҺҘеҸЈиҜ·жұӮж—¶пјҢиҜ·жұӮзҡ„й“ҫжҺҘе§Ӣз»ҲжҳҜlocalhost:yourport/your/api/nameпјҢеңЁwebpack or easyдёӯй…ҚзҪ®proxyжҠҠapiжҺҘеҸЈиҪ¬еҸ‘еҲ°йңҖиҰҒзҡ„еҹҹеҗҚеҚіеҸҜгҖӮ

еҪ“з„¶пјҢиҝҷж ·д№ҹдјҡжңүдёҖдәӣзјәзӮ№пјҡ

  1. жҜҸж¬ЎеҲҮжҚўproxyйңҖиҰҒдҝ®ж”№webpackй…ҚзҪ®пјҢйңҖиҰҒйҮҚеҗҜ(жңүдәӣи„ҡжүӢжһ¶дјҡиҮӘеҠЁйҮҚеҗҜпјҢдҪҶжҳҜд№ҹдјҡиҮӘеҠЁжү“ејҖдёҖдёӘж–°йЎөйқўиҖҢйқһзғӯжӣҙж–°пјҢдёҚжҳҜйӮЈд№Ҳдёқж»‘пјҢеҪ“з„¶д№ҹжңүдәӣи„ҡжүӢжһ¶дјҡзғӯжӣҙж–°)гҖӮ
  2. proxyй…ҚзҪ®иҪ¬еҸ‘ж—¶пјҢеҫҖеҫҖжҳҜжҢүз…§жӯЈеҲҷеҺ»еҢ№й…Қи·Ҝеҫ„пјҢиҖҢжңүдәӣеңәжҷҜпјҢжҲ‘д»¬жңүдәӣжҺҘеҸЈйңҖиҰҒеҲ°жөӢиҜ•еҹҹеҗҚпјҢиҖҢжңүдәӣжҺҘеҸЈйңҖиҰҒеҲ°mockеҹҹеҗҚпјҢе°ұйңҖиҰҒеҒҡдёҖдәӣеҚ•зӢ¬зҡ„еӨ„зҗҶгҖӮеңЁеҗҲд»Јз ҒгҖҒејҖеҸ‘иҝҮзЁӢдёӯеҫҖеҫҖеӨҡжңүдёҚдҫҝгҖӮ

еҖҹеҠ©charlesе®һзҺ°еҹҹеҗҚиҪ¬еҸ‘пјҲжҺЁиҚҗпјү

cookieдёҚжҗәеёҰзҡ„ж №жәҗеңЁдәҺжң¬ең°localhostе’ҢеҹҹеҗҚе’ҢжөӢиҜ•зҺҜеўғзҡ„еҹҹеҗҚдёҚеҗҢпјҢеҜјиҮҙи·Ёеҹҹд»ҺиҖҢcookieдёҚжҗәеёҰпјҢйӮЈд№ҲеҸӘйңҖиҰҒйҖҡиҝҮд»ЈзҗҶе·Ҙе…·е°ҶдёҺжөӢиҜ•зҺҜеўғеҗҢеҹҹзҡ„еҹҹеҗҚд»ЈзҗҶеҲ°жң¬ең°пјҢеҚіеҸҜи§ЈеҶіиҜҘй—®йўҳгҖӮ

з”ұдәҺз¬”иҖ…иҮӘе·ұжҜ”иҫғзҶҹжӮүcharlesпјҢжӯӨеӨ„д»ҘcharlesдёәдҫӢгҖӮ

step1: е®үиЈ…charlesпјҢз„¶еҗҺе®үиЈ…иҜҒд№ҰпјҲдёҚе®үиЈ…иҜҒд№Ұж— жі•иҝӣиЎҢhttpsдҝЎжҒҜзҡ„жҹҘзңӢпјүпјҢgoogle or baiduеҚіеҸҜпјҢжӯӨеӨ„дёҚеҒҡиөҳиҝ°гҖӮ

step2: зӮ№еҮ»tools -> Map Remote -> add, иҝӣиЎҢеҰӮдёӢй…ҚзҪ®пјҲеҒҮи®ҫз«ҜеҸЈжҳҜ3000пјүпјҢе№¶дҝқеӯҳпјҢе№¶еӢҫйҖүenable map remoteеҚіеҸҜгҖӮ

дёҠиҝ°й…ҚзҪ®пјҢдҫҝе°Ҷx x x.163.comдёӢзҡ„иҜ·жұӮпјҢиҪ¬еҸ‘еҲ°дәҶ127.0.0.1:3000з«ҜеҸЈдёӢпјҢжөҸи§ҲеҷЁзӣҙжҺҘжү“ејҖxxx.163.comеҚіеҸҜиҝ”еӣһжң¬ең°3000з«ҜеҸЈдёӢзҡ„йЎөйқўгҖӮжӯӨеӨ„пјҢ163.comд»…д»…жҳҜз¬”иҖ…иҮӘе·ұзҡ„еҹҹеҗҚпјҢе®һйҷ…й…ҚзҪ®ж—¶пјҢдҝқиҜҒжғіиҰҒи®ҝй—®зҡ„жҺҘеҸЈзҡ„дәҢзә§еҹҹеҗҚзӣёеҗҢеҚіеҸҜгҖӮз”ұдәҺcookieзҡ„и·ЁеҹҹеҲӨж–ӯеҸӘеҲ°дәҢзә§еҹҹеҗҚгҖӮеҸҰеӨ–иҝҳиҰҒжіЁж„Ҹhttp е’ҢhttpsпјҢдёӨиҖ…дёҚеҗҢпјҢд№ҹжҳҜдјҡиў«еҲӨе®ҡдёәи·ЁеҹҹпјҢдёҚдјҡжҗәеёҰcookieгҖӮ

WebIDE or CloudIDE

дёҖиҲ¬жқҘи®ІпјҢе…¬еҸёиҮӘе·ұжҗӯе»әзҡ„WEBIDEпјҢйғҪеҹәжң¬дёҺжғіиҰҒи®ҝй—®зҡ„жҺҘеҸЈеҗҢеҹҹпјҢйӮЈд№ҲеңЁејҖеҸ‘ж—¶пјҢз”ұдәҺжң¬иә«е°ұжҳҜеҗҢеҹҹпјҢиҮӘз„¶дёҚеӯҳеңЁи·ЁеҹҹеҗҺcookieж— жі•жҗәеёҰеҜјиҮҙзҷ»еҪ•дёҚдёҠзҡ„й—®йўҳгҖӮ

еҪ“з„¶пјҢwebideзҡ„е»әи®ҫд№ҹйңҖиҰҒдёҖе®ҡзҡ„з§ҜзҙҜгҖӮж„ҹе…ҙи¶Јзҡ„е°Ҹдјҷдјҙз§ҒдҝЎеҠ е…ҘжҲ‘д»¬еӣўйҳҹпјҲдә‘йҹід№җзӣҙж’ӯеӣўйҳҹпјүпјҢдёҖиө·е»әи®ҫwebideе–ҪгҖӮ

е…¶д»–ж–№жЎҲ

дёҠиҝ°еҮ дёӘж–№жЎҲйғҪжҳҜеҸҜиЎҢзҡ„пјҢз¬”иҖ…е…ЁйғЁиҝӣиЎҢиҝҮе®һж“ҚйӘҢиҜҒгҖӮжҺҘдёӢжқҘиҝҳжңүеҮ дёӘж–№жЎҲпјҲдҪҶжҳҜжңӘз»Ҹз¬”иҖ…йӘҢиҜҒпјүпјҡ

ngnixгҖҒwhistleгҖҒfiddlerгҖҒmitmproxyзӯүиҪ¬еҸ‘е·Ҙе…·пјҢеҸҜд»ҘйҖүжӢ©иҪ¬еҸ‘еҹҹеҗҚпјҢиҪ¬еҸ‘жҺҘеҸЈпјҢеҺҹзҗҶдёҠеқҮеҸҜиЎҢ

charlesиҪ¬еҸ‘жҺҘеҸЈпјҢдёҠиҝ°жҳҜиҪ¬еҸ‘еҹҹеҗҚпјҢдҪҝеҫ—cookieеҗҢеҹҹпјҢзҗҶи®әдёҠиҪ¬еҸ‘жҺҘеҸЈд№ҹжІЎжҜӣз—…пјҢдёҚиҝҮжҲ‘д»¬зҡ„жҺҘеҸЈзҺҜеўғеҸҳеҢ–еӨҡз«ҜпјҢй…ҚзҪ®иө·жқҘжҜ”иҫғйә»зғҰпјҢдёҚеҰӮеҹҹеҗҚжқҘзҡ„е®һеңЁ
жңҚеҠЎз«ҜеҒҡдҝ®ж”№пјҢchromeе…Ғи®ёи·ЁеҹҹcookieжҗәеёҰпјҢдҪҶжҳҜcookieйңҖиҰҒж»Ўи¶ідёӨдёӘжқЎд»¶пјҢcookieзҡ„samesiteеұһжҖ§и®ҫзҪ®дёәnone, secureи®ҫзҪ®дёәtrueпјҢзҪ‘з«ҷеҚҸи®®дёәhttpsгҖӮжүҖд»ҘзҗҶи®әдёҠжң¬ең°йңҖиҰҒж”№жҲҗ https://localhost еҗҢж—¶жңҚеҠЎз«Ҝй…ҚеҗҲпјҢе®ҢжҲҗи·Ёеҹҹcookieзҡ„жҗәеёҰпјҲдёӘдәәи§үеҫ—жңҚеҠЎз«ҜдёҚдјҡеҒҡиҝҷдёӘдҝ®ж”№пјҢи·ЁеҹҹcookieжҗәеёҰпјҢи®©csrfж”»еҮ»жқҘзҡ„жӣҙиҪ»жқҫпјү

жңҖеҗҺ

зҘқеӨ§е®¶еҚҮзә§ж„үеҝ«е–ҪпјҒз¬”иҖ…жң¬дәәеұһдәҺзҪ‘жҳ“дә‘йҹід№җзӣҙж’ӯеӣўйҳҹпјҢж°ӣеӣҙеҘҪпјҢжҠҖжңҜзҙ§и·ҹжҪ®жөҒпјҢд№ҹж¬ўиҝҺз§ҒдҝЎжқҘж’©е“ҲгҖӮ

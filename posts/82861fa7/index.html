    2024还在输出技术的波神，这波能反杀吗?

    往者不可谏，来者犹可追 — 《论语·微子》

    大家好，我是芋仔。我正在尝试每周分析一个程序员大佬，到现在已经分析了9个程序员大佬。这期是: 这波能反杀。

    在生活中我们经常会听到这个问题：现在做公众号/掘金/抖音/小红书/b站还来得及吗？

    我们今天的主角，波神—这波能反杀，也在2019年问过这个问题：

    今天就让我们从反杀兄的经历来看看，现在做xx还来得及吗？

    时间线

    反杀兄一般不太分享自己的个人介绍，不过还是在一处荒凉的地方(某雀)，找到了反杀兄的一些个人经历介绍：

    1. 2013年毕业，期间已经参与网页制作，并于腾讯实习
    2. 2013-2017，1年一跳，待了很多公司，放到现在绝对是不稳定的代名词
    3. 2017-2020，某创业项目(保密)
    4. 2020至今，自由职业

    作品一览

    反杀兄的作品数量多且成体系，比较知名的分为以下几个：

    1. Javascript 核心进阶
    2. React hooks 系列
    3. React 知命境系列
    4. React 19系列

    Javascript 核心进阶

    这是波神最早期的作品，2016年开始酝酿。

    在 2017.2～2017.6 月份，在简书上输出了一版(25w的阅读)：

    https://www.jianshu.com/nb/1479424

    在那个时间点可谓是天时占尽，配合上反杀兄的高质高产，以及当时还算技术聚集地的简书，可谓是天时地利人和。

    而这也为他的公众号和个人，都积累了一大波原始粉丝。

    2018.3，书籍《JavaScript核心技术开发解密》出版，也没太推广，销量在当时只能说一般。

    在 2021.4 月份，产出了第二版，上线小专栏平台，为付费项目，定价 100/人。小鹅通，新语后来都有上线，89/人。

    在 2022的3月份，开启b站免费教学视频录制，核心还是javascript核心进阶。

    而 2023.12月份，开启视频公开课，采用直播公开，回放付费的方式进行。

    React hooks 系列

    这一作品，便是在2019年问了：现在认真做公众号还来得及吗之后的首个产出，在这一期间，反杀兄保持了日更的频率。

    产出了 React hooks 系列，当然，罗马并非一日建成，这些内容，其实早在几个月前，简书上就有输出，采用的是付费阅读的模式。

    明确要认真做公众号之后，就开始在公众号上进行日更，产出了：

    1. 超性感的 hooks 系列，2019.11.4 - 2019.11.25
    2. hooks 重构 antd pro 系列，2019.12.3 - 2019.12.6
    3. react + ts 系列，2020.1.1 - 2020.1.14

    这一系列，在公众号是完全免费阅读的。简书上单篇付费，而掘金上则是开放了两篇，剩余引流到公众号。

    之后，主阵地便转移到了公众号上。其余平台更多的是引流和付费需要。

    React 知命境

    初步启动于2021年4月份，但是很快就偃旗息鼓。

    直到 2022年9月份，开始重启，自此不断更新，至今已经有50篇文章左右。

    其中，React哲学为单篇付费，其余文章均直接免费观看。但是会有几个直播教程的回放为付费专属。

    React 19 系列

    2024.5月份开启，大概一个月的时间，完成了比较系统的 react 19学习系列。

    在完成这一系列之后，顺势推出自己的react 19付费小册。

    收入分析

    Javascript核心，第一版真的是重中之重，占据了天时地利人和，积累了大量流量，说是后面所有事情的基础都不为过。而第二版的更新，到 b站的免费视频，付费回放，都顺利转化为了收入来源。

    有了基础粉丝，1v1 定制，高价低量(2w/人)的方式，宣传也很少，收益除了金钱外，授课技巧以及借此沉淀出来的自己的学习论，以及知识的不断更新，也是为后来打下了基础。

    后来hooks系列在简书付费阅读做了尝试后，重启公众号。

    搬运 Javascript基础，hooks 系列，尝试小程序+广告阅读的模式，但是最终以 windows 小程序不支持广告告终。

    又重启第二版的 Javascript 核心，打造一份更优质的付费 Js课程。

    后来 react 知命境，开启单篇一次付费 + 免费直播 + 付费回放的模式，效果还是非常不错的。

    直到近期的 react19 方向的文章，免费优质文章 + 付费沉浸式学习，也是一个非常好的模式。

    而借着这两个低价的优质内容引流，推出相对高价的其他课程，比如鸿蒙，2d可视化等，也会是一个相对较高的转化。

    他山之石，可以攻玉

    纵观反杀兄的一系列探索，能看出来一些值得借鉴的地方，也能看到一些弯路：

    留个人微信，组建私域

    2017年底在发布javascript核心第一版的时候，具备天时地利以及人和。

    而抓住这一波流量的机会，就在于反杀兄在文末有自己的个人微信和公众号，为后来1v1定制服务，2019年认真做公众号，都起到了一个热启动的作用。

    我看到现在很多人做公众号，从0到1积累，真的很慢。但是反杀兄，由于早期已经积累了粉丝，所以公众号直接跳过了冷启动，积累的这个过程。

    输出成体系，垂直发展

    一个体系一个体系的输出，也就是所谓的垂直领域发展。

    越高质量的体系输出，越需要积累。

    越体系的文章，才能在碎片化的阅读中，带来收益的一丝可能性。

    当然，体系化的输出，必然是需要积累的，这中间，即便有真空期，其实也还好，反杀兄的公众号，就有过接近1年的断更期，这并没有对后来的发展带来不利因素。

    反而计划清楚+成体系的输出得到的效果更好。

    高质量免费 + 沉浸式/直播回放付费 + 社群

    反杀兄其实尝试过几个模式：

    最开始简书单篇付费 -> 看广告免费阅读 -> 专栏付费 -> 免费直播引流 -> 高质量免费 + 沉浸式/直播回放付费 + 社群。

    目前来看，高质量免费 + 沉浸式/直播回放 + 社群的付费模式，会是当下的一个比较好的解。

    总结

    反杀兄在17年产出的《Javascript核心》，高质量且契合大环境的内容，配合上文末的引流，顺利积累了一波原始粉丝。

    而之后从单篇付费文章，到看广告免费阅读的探索，都为后来的模式打下基础。

    大量免费的内容：Javascript核心 + react hooks 系列也迅速积累了一定的用户规模。

    之后，免费视频+干货直播+付费回放，借助小专栏/新语/小鹅通等平台，实现了 Javascript 核心的变现。

    而《React 知命境》，凭借着高质量的免费内容，以及极低的付费门槛，又迅速吸引了一波粉丝。

    公开直播+付费回放，输出干货的同时，又提升了付费转化，所谓正循环。

    近期 react19 成体系的 免费输出 + 沉浸式的阶梯付费学习，路径一样，内容也优质，是知识付费的非常值得借鉴的一种模式了。

    而这些，又会可能是1v1定制服务/其他课程的潜在用户，这些1v1以及社群的交流，又可能会带来正向的输出，用互联网黑话来说，就是啪，闭环了！

    回到我们开始的问题：

    现在开始做 xxx 还来得及吗？ 大概率是来得及的，无非就是时间问题。

    波神，这波能反杀吗？ 看收入来源，是反杀成功了，Js核心，React知命境，React19，triple kill无疑了。

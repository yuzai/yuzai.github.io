    object属性名缩写

    当在定义object的时候，如果对属性值采用变量赋值的方法，当变量名和属性名相同时，可以只写一次。

    const name = 'xiaobo'
const person={
  name,
  age:16
};
console.log(person.name);//xiaobo

    判断一个变量是否是数组

    这个方法属于对Array类型的一个扩展

    var obj ={
  name:'xiaobo',
  age:24
}
console.log(Array.isArray(obj));//false
console.log(Array.isArray([1,2,3,[1,2,3]]);//true

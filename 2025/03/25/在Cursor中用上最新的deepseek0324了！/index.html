在Cursor中用上最新的deepseek0324了！

今天一觉醒来，又被国运级产品刷屏了。

这次是 deepseekv3 0324 版本，模型是 3 月 24 晚上开源的，刷屏是 3.25 凌晨，不知道多少自媒体熬夜开更。

在现在的这个信息爆炸，自媒体的话，夸张的嫌疑会高一些。

很多真正关心的信息，反而还是要自己去源头看，道听途说反而一知半解。

发布的源头在这里： https://huggingface.co/deepseek-ai/DeepSeek-V3-0324

一个是各种测试指标的提升，但是都是一些基础指标。

其他更令人注意的是：

前端开发的能力。

不过这一点并没有明确的指标，是否超越 claude 很难评，也不是本文的重点。

本文要介绍的是：如何在 Cursor 中体验下这个最新的模型。

准备工作

  1. Cursor Pro 账号，必须要开 Pro 才可以使用自定义的模型
  2. openRouter 账号，这一步目前该模型是免费的，无需充值

开始

找到 openRouter Model 下的 deepseek0324 模型，其实这里就可以和它聊天了。打到 API tab 下：

创建 API key，记得保存 key。

回到 Cursor -> Preferences -> Cursor Settings -> Model。

点击添加模型，模型名称直接复制 openRouter 中的名称。

然后还是在 Cursor Settings -> Model 下，

拷贝 openRouter 中的 api key，同时修改 下方的 BaseUrl 为：https://openrouter.ai/api/v1即可。

使用

打开这个功能后，很多模型会消失，比如 claude 等模型，同时当选择这些自定义模型时，也不能使用 agent 的能力。同时 tab 补全的能力也用不了。

所以在 Cursor 中，更多的可以使用 edit 或者 ask 能力来和体验这个大模型。切换模型到 deepseekv30324 即可体验。

可以看到，它给自己的定位也是代码专家

十年一剑，卷到极致，躺着赚钱的代码随想录

2024-09-29

大家好，我是芋仔。

今天想跟大家聊一聊程序员carl哥，可能名字听起来陌生，但是如果提到代码随想录，我想应该绝大部分的同学都不陌生。

从2020年末开始每天更新一道算法题，到2021星球创立，再到2022星球反哺输出；从算法训练营的稳定运营，到八股训练营，可谓是一步一个脚印，率先实现了知识”正循环”，说是躺着挣钱也不为过。

但是这个钱，分析完carl哥的经历，说实话，活该他挣。

时间线

我们从卡尔哥不多的个人分享中，能够梳理出如下的时间线：

  1. 2010-2014，某工大 c9 硕士，期间一直有准备 acm，但是并没有战绩
  2. 2014-2019, 毕业后先后在BAT工作
  3. 2019年，公众号周更：发布自己的健身经历 (有一说一，carl哥绝对是程序员里最会健身和街舞的哥哥)。
  4. 2020年3月，开始思变，尝试输出面试心得类，个人成长，算法类文章(周更)
  5. 2020年8月，找到了公众号定位，每天输出一道算法题，开启日更，同时宣传自己私人微信号，拉群组队刷题建立私域。
  6. 2021年，算法日更保持，4月份创建知识星球，同时开启春招秋招社招群创建，连接人，建社群。这一阶段，算法，星球，社群互为配合。
  7. 2021年11月，星球反哺，精华帖输出最强八股第一版，2021年12月，代码随想录书籍输出，半价直接卖爆。
  8. 2022年，公众号日更，主要输出为星球产出：面试经验，职场心得，薪资等等，可谓是反哺循环正式启动。
  9. 2022年5月，星球破万，b 站开更视频：算法140题。
  10. 2022年9月，算法训练营启动，主打打卡陪伴答疑，200一位直接行业最低。
  11. 2023年，公众号日更，主要输出为星球产出：面试经验，职场心得，薪资等等，可谓是反哺循环不断上升。
  12. 2023年8月，卡码网上线，主打自己出题，模拟笔试，线上 oj 等等，同步推出 c++，java 基础课，66地板价。
  13. 2024年，公众号日更，4月启动八股训练营，社招训练营

各平台情况

卡尔哥的所有平台，都是一个账号名：代码随想录。

  1. csdn. 最初的平台是 csdn (我知道很多人吐槽，但是其实carl哥现在还会在 csdn更新，流量说实话还可以)。从 2011年开更，而书籍发布是2021，这也是十年一剑的由来。大概450篇文章，6000粉丝。
  2. github. 2020.5 启动，5w start, 7w粉。
  3. b 站. 2020.7 启动，23w粉，27w赞，核心是代码140题详解
  4. 公众号. 核心阵地，2019 开始，2020.8明确定位，一飞冲天。日更算法，到后来的星球精华帖分享，可谓是星球自生长，内循环就足以躺赚。
  5. 掘金. 非核心，100粉，不维护
  6. 抖音. 4000粉，7000赞
  7. 小红书. 4000粉，2022 加入，算法140题

这里面，核心阵地，就是公众号了。

从周更健身素材，到日更算法题，到日更星球精华帖，也是各个训练营，秋招群，星球的宣传主阵地。

作品

那么，从产品角度来说，carl 哥到底有哪些作品呢？我这边整理了一下：

  1. 代码随想录：配套的网站，pdf, 实体书以及b站配套视频课
  2. 最强八股：知识星球中整理的最强八股，需要加入星球获取。但是carl哥也说了，你可以加进去，下载资料，3天内退出就好了， nobody will care。
  3. 卡码网：提供真题模拟，线上oj，设计模式，语言基础课等内容。
  4. 职场问答：星球精华帖整理的内容。

收入分析

carl 哥，由于网站和视频，都是免费的，所以这块并没有直接带来收益，但是间接的促成了后续的成功。

最初的收入，应该是来自于《代码随想录》书籍，当时据说是引爆双十二。不过目前随着网站内容的不断更新，carl 哥也不建议去买书籍了，毕竟无法及时更新。

而之后，就是知识星球了。

这个星球，一方面，加入费用以及续费，是收入来源。目前星球已经有3w多用户，3w乘上任何一个数字，都是一笔非常可观的收入。

而星球，带来的，还有最强八股文，以及随之而来的八股训练营。

同时更重要的，还产出了非常多的职场，薪资，心路，面经等等资料，这部分也成为carl哥公众号的主要输出内容，而这一部分内容，又进一步奠定了星球的新用户及续费意愿，这个正向循环，生生不息。说是躺赚也不为过。

接下来，就是算法训练营了，打卡答疑，对比动辄上千的价格，200的价格可谓是卷到极致了。而每期人数限制，也从150提高到了180，我想实际报名的人数不会少。

除此之外，八股训练营，相对启动时间较晚，但是创新型的录音打卡方式，同时配上极地的价格，极其公开透明的内容，应该会得到不少人的青睐。

而社招训练营，更多的是一种合作模式，价格较高，具体成果不太好衡量。

崛起技巧

但看事实路径总结下来，大概有这么几个技巧

  1. 找准定位
  2. 建立私域
  3. 扩大影响力
  4. 私域运营反哺公域，公域吸引新粉，形成正循环躺赚

初期：找准定位，转换私域，星球直接热启动 (2020.8 - 2021.4)

可以看出来，carl 哥的第一个关键节点，是2020.8.14在公众号的定位：

但这只是开始，找准了定位，宣传公众号，但是这并不能带来收益。

经过一个月的迭代，carl哥的公众号文末的图片改成如下模式：

加入了个人微信，也是所谓私域的开始，通过打卡，聚集自己的用户群体。

而这，就组成了他的用户群体，随着每一个同学提问的增多，自然而然，就萌生了星球的想法，而这些用户，就是他星球的种子用户，理所当然的启动了星球，直接带来收益。

就像是生孩子，先自己养，孩子大了，有输出了，再反哺公众号，公众号内容又不断壮大星球，所谓自生长，正循环。

中期：出书/秋招/春招群，免费薅云服务器，进一步扩大影响力 (2021.4-2021.12)

有了正向反馈，接下来就是扩大影响力。

上线代码随想录，出书，建群，在2021.10联动敖丙，免费领云服务器，可以说是极速扩张了。

稳定期：星球反哺，职场/面经/薪资/心得/八股，完成自循环，做更多的事 (2021-2023)

算法写完了，公众号日更从星球的精华帖中产出，也基本是大家关心的问题(毕竟别人付费也想知道的内容，多半是大家都关心的)，从而进一步带动更多的人加入星球，产出更多的内容，又带动新的人加入，这个正循环，简直无敌。

而这时候，carl 哥就相对来说空出来了，可以做更多的事情：算法训练营，卡码网，140题视频更新，扩大影响力的同时，利他又利己，可谓是一举夺得。

成熟期：躺着挣钱(2024)

星球反哺公众号，秋招/春招学生一批一批，公众号又反哺星球，简直躺着赚钱(能够打破这个模式的，我想也只有计算机学生的减少了，或者大家都去考公了)

当然，最核心的，还是要先做利他的事情。

总结

纵观carl哥的一路，从2020.8找到定位后，可谓是一飞冲天，迅速崛起。

这中间，坚持不懈的打卡，持续不断的输出，给应届生，社招带来价值，免费硬核的算法内容，是代码随想录的立身之本。

而可能是无意或者有意通过打卡建立的私域，带来的问答，最终促成知识星球的热启动，是carl哥非常非常关键的一步。大概率也是carl哥不收费持续更新算法的底气(毕竟，没有利益的事情，是真的很难坚持下来)。

而之后秋招，春招，社招群，不断的扩大影响力，最终成就了carl哥的中期成功。

接下来星球的运营，精华帖诞生了最强八股以及各种问答，又反而成为了公众号日更的核心内容，而这些内容，又会反过来带给星球新鲜血脉。至此，正循环形成，生生不息，躺着赚钱。

而之后算法140题的讲解视频，算法训练营的诞生，卡码网的出现，给他人带来价值的同时，又能够得到相应的回报，也是这个模式持续运营的动力来源。

最后，就用carl哥的两句话结尾吧：

前方路艰，与君共勉！

你所期盼，终将到来！

js快速排序实现及稳定性分析

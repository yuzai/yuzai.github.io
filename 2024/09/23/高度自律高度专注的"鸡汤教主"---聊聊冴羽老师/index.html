高度自律高度专注的"鸡汤教主"---聊聊冴羽老师

大家好，我是芋仔，这是 分析大佬系列 的第七篇：江湖人称 “鸡汤教主” 的冴羽老师。

分析完他的文章，我想说他是一个三”高”老师：高度自律，高度专注，高度有规划。为什么这么说，请听我慢慢道来。

概况

冴羽老师的主要活动平台有：

  1. 掘金，接近4w粉, 160w阅读，3w赞
  2. 知乎，6000粉，2500赞
  3. 公众号，阅读量基本在 1500 左右
  4. github，1w粉，3w star
  5. 语雀，低调务实优秀中国好青年 社区创始人

从各个平台的情况看，技术文章大阵营还是在掘金上。

而隐藏于掘金背后的低调务实优秀中国好青年社群，从号称千人打卡，持续了两年半的各种线上线下活动的高质量自律社群，可能就止步在了 2024的上半年，着实让人叹息。

系列文章分析

分析冴羽老师的文章，非常简单明了：

  1. 没有私人类的文章
  2. 都是以系列为主，集中输出，不会一篇这个系列，一篇那个系列。极度专注极度有规划，遇到反响不好，立即止步，简直就是行走的自律”鸡汤”标杆。

2017 javascript 双专题直接登顶

javascript 深入专题，篇均点赞 200+，总结篇直接 3500+赞，这个赞基本是掘金天花板了。工作一两年的，非常适合用来面试准备，做面试官的也可以考虑借鉴进行深入提问。不要花钱去报班了，这个免费资源绝对足够。

ps: 还有一个有趣的深入，同样是这个系列，2020发布的，其实关注度就不高了，所谓天时地利人和。只能说：2020，天时已过。

javascript 专题系列，篇均点赞 200+，总结篇 2800+ 赞，别花钱去报班了，学不会 js？看这个系列绝对没错。

2018 underscore 专题略逊一筹，ES6 系列再起飞

underscore 系列的灵感来自于 js 专题中的一些文章，衍生出来的话题，整体数据还可以，但是绝对谈不上爆。

之后的 es6专题 又回归巅峰，总结篇 2000+ 赞。

2019-2023 消失的两年

2019 年和 2023 年并无技术系列文章产出。

2020年集中在面试文章，数量不多，但是反响非常不错。

2021 集中在 Typescript 系列。

2021 年其实 ts 还是比较火的，但是冴羽老师的文章集中在官方文档的翻译，并无实质性的内容产出，所以数据一般。这里可以提一提我自己的 typescript 类型体操通关手册，有需要自取。

这里就体现出来冴羽老师的一个强大之处，有规划，但是数据一般，就中止。参考 冴羽ts文档。进阶和实战篇，估计将会一直停留在准备中了。ps: 证书也过期啦。

2022 年，冴羽老师的精力就从 ts 转向了 博客搭建系列 和 答读者问系列。

博客搭建系列，反响一般，但是内容非常适合初学者搭建自己的博客，非常推荐需要搭建博客的同学学习。

答读者问数据还不错，这一系列，就是被称为”鸡汤教主”的来源了。还算是比较受欢迎的(如果说别人来讲这个鸡汤，我可能是不服气的，但是仔细看冴羽老师的事迹，有规划，迅速的输出系列文章，谋定后动的自律精神，绝对是行走鸡汤，与其说是鸡汤教主，不如说是鸡汤本汤)。

之后又开启了 React 系列。

耗时一月，反响只能说还不错，果断收手。

2023 技术文章上又消失了一年，这一年，是活跃的低调务实优秀中国好青年社群的群主。

2024 next 系列+小册，正式变现

数据还不错，同时推出了自己的 next 小册。

这一把可谓是终于进行了知识变现。有强大的粉丝基础+以往的口碑，这个小册还是非常受欢迎的。赛道也比较好，next.js 确实没有中文文档，同时竞品也不多。说是卖爆也不为过。

回头一看，next 系列文章和 next 小册互为配合，可谓是不鸣则已，一鸣惊人：极度有规划，极度专注。

茶话会？低调务实优秀中国好青年？21天打卡学习营？

这在现在看来，着实是个神秘组织。

上千名程序员一起早起，打卡，写作？还有 21天打卡营？收费的吧哈哈？

我想你一定和我有同样的疑惑，这是程序员能干的事？但是实际上，这个事情还真做起来了。

从 2022 年2月份启动，而3月份，冴羽老师就启动了答读者问，之后也是持续输出了小半年，这才有了”鸡汤教主”的称号。

这里面就不得不提到 21天学习训练营。

相信听到训练营，你也一定和我一样，想到了收费，没错，确实是收费的，从社群收入明细中我看到，好像只有 60，而且完成打卡，会全额返还，所以别说割韭菜了，60也能算？比起付出的辛苦，这简直是妥妥的赔本生意。

而整个社群，还是很活跃的，有打卡小程序，有群聊机器人，也有很多人自发购买掘金小册返佣增加社群收入。

2023年，冴羽老师虽然没有写技术文章，但是组织了12场茶话会，内容也基本全部开放出来，当然也有微信付费专栏，但是说实话，愿者上钩，不愿意买的，看视频完全足够。内容呢，全部都是个人管理上的。

说实话，这个个人管理，冴羽老师写，绝对当之无愧，因为他自己，就是这套理论的践行者。

只是天下熙熙，皆为利来；天下攘攘，皆为利往。

没有利益收入的组织，终归可能就是尘归尘，土归土。

收入来源分析

基本副业收益，应该是都来自于 next 小册了。

这些作品，火归火，但是转换为收益的，其实并不算多。

最后

冴羽老师，绝对是高度专注，高度自律，高度有规划的一个人(盲猜mbti人格是 INTJ 紫老头)。

高度专注，体现在系列文章的持续输出，这个系列不结束，不开新坑。

高度自律，体现在社群规划的一步一步产出，早起，打卡，简直是”鸡汤本汤”。

高度有规划，体现在系列文章的输出，反馈好，就继续，不好，就换赛道。而next 系列输出和小册的配合，当真是不写则已，一写就是一个系列。

当然，大佬也有没填好的坑：比如 ts 的进阶和实战篇，并未完结但已经过去一两年的掘金专栏。估计也不会填了，毕竟没有收益的事情，是真的很难坚持，又或者说，这种事其实是没有必要坚持的。

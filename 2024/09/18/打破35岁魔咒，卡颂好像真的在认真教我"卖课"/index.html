打破35岁魔咒，卡颂好像真的在认真教我"卖课"

大家好，我是程序员芋仔。

之前几篇分析了鱼皮，java3y，敖丙，光神，这次也终归是轮到了卡颂老师(欢迎去我同名公众号: 程序员芋仔，查看合集哦)。

概览

卡颂老师的主要平台 掘金/知乎/b站/微信公众号。微信公众号的阅读量基本在 2000 左右(收入在 20-30w)。一些非技术类文章，会更多的发布在微信公众号(发布掘金看的人也不多)。

掘金的情况如下：

时间线

从卡颂老师为数不多的年终总结中，可以大概看出他的一个个人脉络，也能判断出他已经破除了35岁的魔咒(如果35裁员是魔咒的话，那当我直接辞职搞自由职业，阁下如何应对呢？)。

  1. 2020年，开源了电子书 react技术揭秘，同时还有 从0到1实现react，也录制了思否视频课程，迅速建立起一个 2000 人的 react 学习社群，瞬间有了副业，收入在 6000 - 10000 之间，不多，但是贵在被动收入。年终总结：大佬那么多，为什么不能是我，可谓是意气风发。

  2. 2021年持续输出 react 原理，诸多考虑下，选择离职。与此同时，公众号也开始发一些面试视频。

  3. 2022年，躺平写书，开源节流，被动收入真的很香。

  4. 2023年，ai + 前端 + react，持续输出 react 干货

  5. 2024年，渐渐不聊技术了。开始转型，教他人挖掘优势，建立副业体系。

掘金文章分析

卡颂老师，文章总数是 209，其从 2020 开始发力，直到 2024，文章点赞比逐年增高。

2020 深入react源码分析，幽默干货，崛起的一年

卡颂老师的文章基本都是围绕 react 源码分析来，这里也会顺带推广自己的开源电子书和视频课程。

这里面有一个非常有趣的点，就是大佬也需要标题党。可以看出来，排名最高的文章，就是 阿姨我不想努力了，600+赞，在掘金上绝对算得上高赞文章了。

除此之外，干货也是高赞的一个来源。

ps: 你会发现，也有很多文章赞也只有个位数。所以刚来掘金的同学不要气馁哈，大佬的文章点赞都有个位数的。

2021 不温不火，持续输出

可以看到，点赞不温不火。这一年的文章，还是围绕 react 展开，只能说，有深度，但是在掘金，可能面试和人生感悟的文章更受欢迎。

2022 副业有成，躺平出书

人生分享 + react 干货构成了主旋律，也开始一些新的探索，比如 rome，交互式文章等。

更详细的数据，可以看看卡颂老师自己的分享：卡颂：30岁程序员选择躺平

这一年，卡颂老师正式离职开启自由职业，顺带写了本react的书。可谓是真正打破了35岁”魔咒”。

2023 借力 AI，文章爆赞

可以看到，在 AI 浪潮下，卡颂老师第一个跟进，相关文章也是水涨船高。不过干货依旧，这一年的文章数据可谓是非常好看。

2024 技术技术，不如找找自己的优势

这一年卡颂老师的技术文章明显变少，原因可以看看他自己的文章 为什么我挺久没写技术文章了，原因很现实。

但是这部分文章，可能才是当下这个环境下我们所需要的。

不过卡颂老师明显考虑到了掘金这个平台对于非技术文章的不喜，所以类似这种文章，公众号会更多一些，我们在公众号篇分析。

公众号文章分析

公众号的脉络更为简单，一方面，掘金上的技术类文章，基本公众号上都有。

除此之外，卡颂老师还研究过：

  1. 30s 学会 js 系列
  2. 30s 学会 css 系列
  3. 面试视频号
  4. 关于卖课，副业的思考

这里面，持续更新的，除了技术，就是关于卖课，关于程序员副业的思考。而这一类文章，从 2023 就开始出现，很多都只在公众号上发。

我做了一个简单的分类：

  1. 卡颂老师教你做独立开发
  2. 卡颂老师教你卖课
  3. 卡颂老师给你治精神内耗
  4. 卡颂教你做社群
  5. 卡颂收入分享

可以看出来，2024，互联网岗位减少，卡颂老师，可能是真的在教程序员开展第二曲线。

最后

从卡颂老师的经历来看，核心产品就是 react源码分析，围绕这一主题，依次开源了 react技术揭秘, 从0到1实现react，核心在于，开源免费，自己学不收费，但是如果觉得自学不行，就买收费的视频课。

这一策略，快速建立起了学习社群，建立了影响力。

卡颂老师有一个核心观点：流量是杠杆，而不是红利。先有产品，也就是开发自己的课程，然后借助流量对其进行放大。

在有了稳定的副业收入之后，又写了本书，进一步扩大自己的影响力。

有了稳定的收入来源，也就有了更多的思考空间，卡颂老师沉淀了自己的第二曲线理论，可以看看公众号：Symbol第二曲线。

核心在于以自己为例，提出了非标品的概念，也沉淀了一套自己维系社群的理论。建议打算出课程的都看一看。

最后，如果说分析完敖丙，我有一些绝望，分析完光神，我看到一道光，那么分析完卡颂老师，我似乎能看到一条自己的第二曲线路径。

就用卡颂老师的一句话结尾吧：大佬那么多，凭什么不能是我！

对了，欢迎大家关注我的公众号：程序员芋仔，获取最新文章和分析大佬合集哦！
